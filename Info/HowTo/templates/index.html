Python and Django

-Web Development-

Have you ever wanted to learn how to code? Now you can! With this beginners guide to web development in Python, you will be learning the basics of python including but not limited to, data types, conditional statments, control flow, logical operators, string methods and much more. You will also learn about Django. Django is a web development and rest api framework for Python. We will go over how to install the lastest version of Python and Django, How to set up and activate a virtual environment, the django-admin commands to start your first Python and Django project and how to start a localhost server.
